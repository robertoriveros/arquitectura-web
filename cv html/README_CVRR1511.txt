*PROYECTO : curriculum vitae en html 5.
- el proyecto esta contenido en una carpeta maestra, la misma se compone a su vez de 3 elementos:
* carpeta - css : contiene las hojas de estilo.
* carpeta - img : contiene las imagenes del proyecto.
* archivo .html : contiene el codigo que genera el sitio web en html.

EL ARCHIVO HTML 5 ESTA DIVIDO EN DOS GRANDES GRUPOS:
*RESUME CONTENT : CONTIENE LOS ELEMENTOS DE LA IZQUIERDA (FOTO, SKILLS,PROFILE).
* RESUME RIGHT : CONTIENE LOS DETALLES DE EDUCACION, EXPERIENCIA LABORAL, ETC.
